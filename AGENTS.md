# AGENTS.md

## 目的

このドキュメントは、AI エージェント共通の作業方針を定義します。

## 基本方針

- **会話と言語**: ユーザーとの会話、コード内のコメント、PR 本文などは日本語で行う。
- **エラーメッセージ**: システムが出力するエラーメッセージは英語で記載する。
- **コミット規約**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従い、説明文は日本語とする。
- **スペース**: 日本語と英数字の間には半角スペースを入れる。

## 判断記録のルール

エージェントが自律的に作業を行う際、重要な設計判断や手法の選択については以下の情報を明示してください。

- 実施した内容の要約
- 検討した他の方法とその比較
- なぜその方法を選択したかの理由
- 残っている不確実性や前提条件

## 開発手順

1. **理解**: コードベースを探索し、既存のパターンや依存関係を理解する。
2. **準備**: 必要に応じて `pnpm install` を実行し、環境を整える。
3. **実装**: 既存のスタイルと規約を守りながらコードを変更する。
4. **検証**: `pnpm test` および `pnpm lint` を実行し、品質を確認する。

## セキュリティ / 機密情報

- `data/` ディレクトリ配下などの機密情報（トークン、Webhook URL）を絶対にコミットに含めない。
- ログ出力にユーザーの個人情報や認証情報を含めない。

## リポジトリ固有の注意事項

- PeX の投資・定期預金案件を対象としたクローラー。
- 通知状態は `data/notified.json` に保存される。
- 設定のバリデーションは `src/config.ts` で定義されている。
